
# Usage

run main file

```bash
    $ ./run <DIMENSION> <SCRAMBLE>
DIMENSION = cube dimension
SCRAMBLE = number of scramble moves
```

# Summary
I built this project my freshman year in college, so it's not top quality code, but it works! It works for any dimension size 3 or greater (doesn't work for 2x2 cubes).

I used the algorithm I know (no speed cubing here, vanilla beginners rubik's cube algorithms).
